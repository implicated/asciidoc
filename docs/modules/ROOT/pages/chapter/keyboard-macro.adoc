== Keyboard Macro

|===
|Shortcut |Purpose

|kbd:[A]
|This a key

|kbd:[⌃,⇧,⌘,⌫,⌦,⇪,⇥,⎋,⏎,⌥,fn,↑,↓,←,→]
|Technical Symbols


|kbd:[F11]
|Toggle full screen

|kbd:[Ctrl+T]
|Open a new tab

|kbd:[Ctrl+Shift+N]
|New incognito window

|kbd:[\ ]
|Used to escape characters

|kbd:[Ctrl+\]]
|Jump to keyword

|kbd:[Ctrl++]
|Increase zoom
|===
